# Licensing Legal/EULA Check (2026-04-02)

## Scope

Confirm licensing controls and anti-abuse enforcement are reflected in contractual terms.

## Checklist

- [x] License activation requirement documented for all installations.
- [x] Seat limits and overage behavior documented.
- [x] Billing enforcement (`grace` -> `suspended`) documented.
- [x] Device transfer/deactivation policy documented.
- [x] Emergency lock/token revoke/force reauth authority documented.
- [x] Audit logging and retention expectations documented.
- [x] Manual payment verification and second-approval controls documented.
- [x] Fraud/abuse response process referenced.

## Required Legal Follow-Up Before Production Contract Freeze

1. Confirm jurisdiction-specific language for remote disable/lock capabilities.
2. Confirm customer notice language for suspension/revocation actions.
3. Confirm acceptable-use and anti-circumvention clauses align with enforcement telemetry.
4. Confirm data-processing disclosures for source fingerprint and security-event metadata.
